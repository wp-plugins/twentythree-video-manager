== Description ==

This plugin creates a link between your http://www.23video.com/ site and a wordpress site. 

Simply provide the plugin with the URL of the video site and a tag to search for. 

The plugin will then search your video site for that tag and create a gallery on your wordpress site, by using the [23video] shortcode

== Installation ==

Follow these instructions to install the plugin

1. Upload `23Video-plugin to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to 'Settings' and then '23 Video' and fill in the necessary details
1. Use `[23video]` in your pages/post to display the videos
